# 📰 BBC News Text Classifier

##  Overview
This project classifies BBC news articles into categories such as business, sport, politics, tech, and entertainment using Natural Language Processing (NLP) techniques.

##  Features
- Text preprocessing (cleaning + lemmatization)
- CountVectorizer with n-grams
- Multinomial Naive Bayes model
- Model saving using pickle
- Modular code structure
